#### Run in your local machine:

  

- Clone the repository:

  

```sh

  

git clone  https://github.com/bilal-08/weather-app

  

cd weather-app

  

```

  

- Install the dependencies:

  

```sh

  

npm i

  

```

  

- Running the webapp:

  

```sh

  

npm run  dev

  

```
